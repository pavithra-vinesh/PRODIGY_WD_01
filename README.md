# PRODIGY_WD_01
Interactive Navigation Menu

PROJECT OVERVIEW

This project demonstrates a responsive and interactive navigation menu built using HTML, CSS, and JavaScript. The navigation bar remains fixed at the top of the page and changes its appearance when the user scrolls or hovers over menu items, creating a modern and engaging user experience.


FEATURES

Fixed navigation bar visible at all times.

Smooth hover effects on menu items.

Dynamic style changes when scrolling.

Clean and responsive design.

Easy navigation between page sections.

Built using pure HTML, CSS, and JavaScript.

TECHNOLOGIES USED 

HTML
CSS
JavaScript

HOW IT WORKS

HTML

Creates the navigation bar and webpage sections.
Defines menu links such as Home, About, Services, and Contact.

CSS

Styles the navigation menu.
Adds colors, spacing, hover effects, and transitions.
Keeps the navigation bar fixed at the top.

JavaScript

Detects page scrolling.
Dynamically changes the navigation bar's style when the user scrolls.
Enhances user interaction and visual feedback.
