# 🚀 Interactive Navigation Menu

## 📌 Project Overview

This project is a responsive and interactive navigation menu built using HTML, CSS, and JavaScript. The navigation bar remains fixed at the top of the page and changes its appearance when the user scrolls or hovers over menu items.

The project demonstrates how to create a modern and user-friendly navigation system for websites.

---

## ✨ Features

- Fixed navigation bar
- Smooth hover effects on menu items
- Dynamic color/style changes on scroll
- Responsive layout
- Easy navigation between sections
- Clean and modern user interface

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

## 🏗 How the Project Works

### HTML
- Creates the structure of the navigation menu.
- Defines sections such as Home, About, Services, and Contact.

### CSS
- Styles the navigation bar and page sections.
- Adds colors, spacing, hover effects, and transitions.
- Keeps the navigation menu fixed at the top of the page.

### JavaScript
- Detects page scrolling.
- Changes the navigation bar style when the user scrolls.
- Enhances interactivity and user experience.

## ▶️ How to Run the Project

### Method 1: Open Directly

1. Download the project files.
2. Open `index.html` in a web browser.

### Method 2: Using VS Code

1. Open the project folder in VS Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.
